## Dictionaries

Like lists, dictionaries are a type of collection which can contain different [types](/1%20Python%20Basics/1.2%20Types.md#types). Instead of having integer indexes as [lists](/2%20Python%20Data%20Structures/2.1%20Lists%20and%20Tuples.md) do, dictionaries have keys and values. Keys are analagous to list indexes, but do not need to be integers.  
*As of Python version `3.7`, dictionaries are ordered. In Python `3.6` and earlier, dictionaries are unordered.*

Dictionaries are defined using curley brackets `{}`, Each key and value pair is separated by a comma.  
The keys are the first elements; they must be immutable and unique. Each key is followed by a value separated by a colon.  
The values can be anything, including simple types like `int`, `float`, `str`, or `bool`; or complex types like `list`, `set`, `tuple`, or even `dict`.

**Example Dictionary:**  
```py
dict = {
    "name": "Alice",
    "age": 30,
    "hobbies": ["reading", "cycling", "gardening"],
    "is_student": False,
    "scores": {"math": 90, "science": 85, "english": 88},
}

```
The table below displays the relationship between keys and values, note that nesting is permitted in dictionaries:  

|     Keys     |                  Values                    |       Type        |
|--------------|--------------------------------------------|-------------------|
|     name     |                   Alice                    |      String       |
|     age      |                    30                      |      Integer      |
|   hobbies    |    ["reading", "cycling", "gardening"]     |       List        |
|  is_student  |                  False                     |      Boolean      |
|    scores    | {"math": 90, "science": 85, "english": 88} | Nested Dictionary |

Keys may be strings or any immutable object, such as a tuple.


## Dictionary Operations
A number of operations can be performed on dictionaries in order to manage and manipulate the data, these are described below.

### Retrieve Values
Values can be retrieved by specifying key names; 
```py
dict = {
    "name": "Alice",
    "age": 30,
    "hobbies": ["reading", "cycling", "gardening"],
    "is_student": False,
    "scores": {"math": 90, "science": 85, "english": 88},
}
dict['name']
dict['hobbies']
```

Will output:
```
'Alice'
['reading', 'cycling', 'gardening']
```

Alternatively all values in a dictionary can be retrieved using the `values()` method:
```py
dict = {
    "name": "Alice",
    "age": 30,
    "hobbies": ["reading", "cycling", "gardening"],
    "is_student": False,
    "scores": {"math": 90, "science": 85, "english": 88},
}
dict.values()
```

Will output:
```
dict_values(['Alice', 30, ['reading', 'cycling', 'gardening'], False, {'math': 90, 'science': 85, 'english': 88}])
```

### Retrieve Keys
The keys of a dictionary can be retrieved using the `keys()` method thi will retrieve allthe keys of the dictionary.
```py
dict = {
    "name": "Alice",
    "age": 30,
    "hobbies": ["reading", "cycling", "gardening"],
    "is_student": False,
    "scores": {"math": 90, "science": 85, "english": 88},
}
dict.keys()
```

Will output:
```
dict_keys(['name', 'age', 'hobbies', 'is_student', 'scores'])
```

### Add Keys and Values
There are a number of ways to add values to a key in dictionaries; these will depend on the type of values the key contains:
* Strings, integers, booleans: Direct assignment is straightforward.
* Lists: Use .append() for a single value or .extend() for multiple values.
* Dictionaries: Use .update() to add or modify key-value pairs.
* Tuples: Since theyâ€™re immutable, you need to create a new tuple and reassign it.

#### Direct Assignment

```py
dict = {
    "name": "Alice",
    "age": 30,
    "hobbies": ["reading", "cycling", "gardening"],
    "is_student": False,
    "scores": {"math": 90, "science": 85, "english": 88},
}

dict["name"] = "Bob" # Replace "Alice" with "Bob"
dict["name"]
dict["study_years"] = (2023, 2024) # Add new key and (tuple) values
dict["study_years"] 
```

Will output:
```
'Bob'
(2023, 2024)
```

Values can be appended to a key using the `append()` method;
```py
dict = {
    "name": "Alice",
    "age": 30,
    "hobbies": ["reading", "cycling", "gardening"],
    "is_student": False,
    "scores": {"math": 90, "science": 85, "english": 88},
}
dict["hobbies"].append("knitting")
dict["hobbies"]
```

Will output:
```
['reading', 'cycling', 'gardening', 'knitting']
```

It should be noted that the `append()` method will only work with list entries. To append a value to `"name"`, it first needs to be converted to a list:
```py
# Example dictionary
dict = {
    "name": "Alice",
    "age": 30,
    "hobbies": ["reading", "cycling", "gardening"],
    "is_student": False,
    "scores": {"math": 90, "science": 85, "english": 88},
}

dict["name"] = [dict["name"]] # Convert existing value to a list
dict["name"].append("Bob") # Append the new name
print(dict["name"])
```

Will output:
```
['Alice', 'Bob']
```
